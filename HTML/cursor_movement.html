Cursor movement

 

h move cursor left
j move cursor down
k move cursor up
l move cursor right
H move to top of screen
M move to middle of screen
L move to bottom of screen
<Ctrl>b move back one full screen
<Ctrl>f move forward one full screen
<Ctrl>u move back one harf screen
<Ctrl>d move forward one harf screen
<Ctrl>e move screen down one line (without moving cursor)
<Ctrl>y move screen up one line (without moving cursor)
 z<Enter> top cursor on screen
zt top cursor on screen
zz centor cursor on screen
z- bottom cursor on screen
gg go to the first line of the document
G go to the last line of the document
:5 go to line 5
5G go to line 5
0 jump to the start of line
$ jump to the end of line
^ jump to the first non-blank character of the line
g jump to the last non-blank character of the line
w jump forwards to the stat of a word
e jump forwards to the end of a word
b jump backwords to the end of a word
{ jump to next paragraph (or function/block, when editing code)
} jump to previous paragraph (or function/block, when editing code)
% move to matching character (default supported pairs: '()', '{}', '[]')
f jump to next occurrence of character x
; repeat previous f movement
